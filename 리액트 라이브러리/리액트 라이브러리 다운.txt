리액트 라이브러리 다운
npm install @mui/x-date-pickers - MUI 날짜 선택기
npm install @mui/material @emotion/react @emotion/styled - MUI 핵심 패키지, 스타일링
npm install @mui/icons-material - MUI 아이콘 패키지

npm install react-modal - 모달 설치
npm install react-datepicker react-modal - 달력 선택
npm install react-multi-date-picker - 멀티 선택
